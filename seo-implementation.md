## 7. TECHNICAL SEO

### Viewport Meta Tag

- `width=device-width, initial-scale=1.0`
- Ensures mobile responsiveness
- Improves mobile search rankings

### Character Encoding

- UTF-8 (supports Malay and English text)
- Proper encoding for international audience

### Robots.txt File

- Allows all user-agents to crawl
- Specifies sitemap location
- Sets crawl delay of 1 second
- **Location:** /robots.txt

### Sitemap.xml File

- Lists all important pages
- Includes image references
- Last modified dates
- Priority and change frequency
- **Location:** /sitemap.xml
